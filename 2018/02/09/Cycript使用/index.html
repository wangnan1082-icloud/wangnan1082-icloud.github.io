Cycript使用

Posted on 2018-02-09 Edited on 2023-05-30 In Cycript

基本使用

  • 运行cycript
  • 退出cycript

Control+D，来退出Cycript

  • 查看当前机器的所有进程

使用命令 ps -e，之后可以看到下面的 QQMusic。

  • 确定进程

ps aux | grep QQMusic

  • Cycript钩住QQ音乐进程

cycript -p 45025

  • 打印视图层级

UIApp.keyWindow.recursiveDescription().toString()

  • choose传递一个类，可以在内存中找出属于这个类的所有对象

choose(UITextField)

进阶使用

使用 MJ 的 MJCript，来方便使用打印。具体使用方法可查看链接。

下载好 mjcript.cy，

// 复制到越狱手机

scp /Users/Downloads/mjcript-master/mjcript.cy root@192.168.32.125:/usr/lib/cycript0.9

// 引入

@import mjcript

// 开始使用

MJRootVc() // 根控制器

MJFrontVc() // 最前面的ViewConroller

MJFrontVcSubViews() // 递归打印最上层UIViewController view的层级结构

MJSubviews(#0x16293c00)

MJInstanceMethods() // 打印所有的对象方法

简单实践

  • 更改手机锁屏时滑动解锁文字

钩住 SpringBoard 进程：

cycript -p SpringBoard

查看View层级关系

UIApp.keyWindow.recursiveDescription()

获取对象：

var manager = SBLockScreenManager.sharedInstance()

var controller = [manager lockScreenViewController]

var view = controller.lockScreenView()

设置锁屏文字：

[view setCustomSlideToUnlockText: “Wangnan”]
