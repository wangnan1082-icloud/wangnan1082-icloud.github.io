使用 OHHTTPStubs

发表于 2017-12-25 | 分类于 OHHTTPStubs

OHHTTPStubs 可以在不需要后端 API 的情况下，在本地对 HTTP 请求进行拦截，返回想要的 json 数据，可以用来编写网络请求的单元测试。

基本用法

拦截域名为http://www.test.com/test的HTTP请求，并返回本地testData.json文件中的数据，并可以设置网络状态码为200，请求头格式，以及请求时间和响应时间。虽然这个 URL 并不存在，但是这里的代码通过 HTTP stub 成功地模拟了 HTTP 响应。
